破解软件值多少？

发表于2024-10-18|更新于2024-10-18|Miscellaneous
|字数总计:1.1k|阅读时长:3分钟|阅读量:|评论数:

Windows11

​ Windows 是微软公司（Microsoft）开发的一系列操作系统。自从 1985 年推出 Windows 1.0 以来，Windows 系统已经成为全球最广泛使用的个人电脑操作系统之一。Windows 11家庭版正版授权费用 1080。

Microsoft 365

​ Office 套件 Word、Excel、PowerPoint 大家应该也都用过，之前一直以为“免费” Windows 里自带的 Office 套件理应也是免费的。后面才知道，它也是收费的，而且跟 Windows 系统是分开收费的。Microsoft 365 个人版正版授权每年费用 398。

Photoshop

​ Adobe Photoshop 是一款广泛使用的图像和照片编辑软件，它提供了一系列强大的工具和功能，用于创建和修改数字图像。Photoshop 广泛应用于广告、平面设计、摄影、三维渲染、网页设计等多个领域。Photoshop 一般设计师都在使用，但我们开发人员偶尔也会自己安装来用。Adobe 旗舰店显示，个人版中国摄影计划正版授权费用 978/年，手机端打开是 848/年。团队版：价格为 6942 元人民币/年。企业版：价格为 11409 元人民币/年。

SecureCRT

​ SecureCRT 是一款专业的终端仿真程序，它支持多种网络协议，包括但不限于 SSH（SSH1和SSH2）、Telnet、rlogin 以及串行协议等。SecureCRT 被广泛用于连接远程系统，如 Windows、UNIX 和 VMS，并且提供了高级的会话管理功能和数据安全特性。官方显示价格 119 美金/年，将近人民币 800 多每年。

PyCharm

​ PyCharm 是由 JetBrains 公司开发的一款流行的 Python 集成开发环境（IDE），广泛用于 Python 编程。PyCharm 两个版本：专业版（Professional）和社区版（Community）。专业版个人使用第一年费用 99 刀，人民币差不多700多。

IntelliJ IDEA

​ IntelliJ IDEA 也是由 JetBrains 公司开发的一款功能强大的集成开发环境（IDE），主要用于 Java 和 Kotlin 语言的开发。专业版个人使用第一年费用 169 刀，人民币 1200 多。JetBrains 开发了很多款流行的集成开发环境（IDE）和其他工具，如果买他的全家桶价格还更划算些。专业版个人使用第一年费用 289 刀，人民币 2000 多，比起一个个购买划算很多了。

Navicat for MySQL

​ Navicat for MySQL 是一款专为 MySQL 和 MariaDB 数据库设计的数据库管理和开发工具。它通过提供一个直观的图形用户界面（GUI），简化了数据库的管理、开发和维护工作。非商业版本一年收费 599，永久许可证 1099。Navicat Premium 是全家桶数据库库管理数据，支持多种数据库系统，包括 MySQL、SQLite、Oracle、MariaDB、SQL Server、PostgreSQL 和 MongoDB 等多种数据库。非商业版本一年收费 2899，永久许可证 5699。

Transmit

​ Transmit 是一款由 Panic Inc. 开发的 FTP（文件传输协议）客户端软件，提供了一种用户友好的方式来传输和管理文件，主要用于 macOS 平台。收费 45 刀。

FlashFXP

​ FlashFXP 是Windows 平台的 FTP 客户端，支持 FTP、FTPS、SFTP 等协议，提供安全、可靠、有效率的文件传输。FlashFXP 界面友好，对新用户来说易于上手，支持多文件夹选择文件和文件夹比较功能。目前折扣价格 16 刀，人民币要110 多。

Termius

​ Termius 是一款流行的跨平台 SSH 和 Telnet 客户端，用于远程访问、控制服务器和管理网络设备。Termius 是免费软件，但提供可选的付费版本，称为 Termius Pro，它包括了一些额外的功能，如端口转发、标签自定义、同步等。收费版本从每月 10 刀起。

ChatGPT Plus

​ ChatGPT 是一个由 OpenAI 开发的大型语言模型，现在我不知道有没人破解，这个应该挺难。Plus 版本每个月 20 刀，140多人民币。

VMware 虚拟机

​ VMware 收费的产品也很多，我们个人常用的是 VMware Workstation。VMware Workstation 是一款桌面虚拟化软件，允许用户在单一的物理机器上运行多个操作系统。费用 149 刀，折合人民币也要 1000 多。
